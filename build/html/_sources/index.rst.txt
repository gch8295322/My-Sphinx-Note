简介
==========
这里是郭禹淳的博客，记录学习生活中遇到的各种问题及解决方法。更新该博客时，每次编译时记得清掉build再make，即：
::
   ./make clean
   ./make html
然后更新到Github上，再去Read the Docs上手动编译下，或者过段时间它会自己编译。
::
   git add .
   git commit -m "commit"
   git push -u origin master
